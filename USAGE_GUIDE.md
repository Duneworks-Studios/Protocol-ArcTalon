## üîê Puzzle Solutions & Walkthroughs

### Puzzle 1: Origin Log - AuraGenesis
**Difficulty**: Easy | **Type**: Base64 Decryption

**Challenge**: Decode `QVVSQURZTkFNSUNT`

**How to solve**:
1. Copy the Base64 string
2. Use an online Base64 decoder (e.g., base64decode.org)
3. Paste and decode
4. Enter the result: `AURADYNAMICS`

**What you learn**: The original name of Duneworks Studios

---

### Puzzle 2: Transmission - Revive Era
**Difficulty**: Easy | **Type**: Morse Code

**Challenge**: Decode `.-.. . --. .- -.-. -.--`

**How to solve**:
1. Use a Morse code translator
2. Input the dots and dashes
3. Get the decoded word
4. Enter: `LEGACY`

**What you learn**: A heartfelt message to the Revive Era friends

---

### Puzzle 3: Memory Node - D.Lee.Buckley
**Difficulty**: Medium | **Type**: Fragment Reconstruction

**Challenge**: Place fragments in correct order

**Solution order**:
1. Slot 1: "Daniel Lee Buckley"
2. Slot 2: "60-100M USD annually"
3. Slot 3: "mirror that success"
4. Slot 4: "only saw me as a wallet"

**How to solve**: Click each fragment and assign it to the matching slot

**What you learn**: Daniel's personal background and motivation

---

### Puzzle 4: Incident Log - Ownership Transfer
**Difficulty**: Medium | **Type**: Data Corruption Repair

**Challenge**: Fix 4 corrupted blocks

**Solutions**:
1. Block 1: `2@25` ‚Üí `2025`
2. Block 2: `Sp!nd##l` ‚Üí `SPINDEEL`
3. Block 3: `m!st@k#` ‚Üí `MISTAKE`
4. Block 4: `R#v!v@l` ‚Üí `REVIVAL`

**How to solve**: Remove corruption symbols (@, !, #) and restore original text

**What you learn**: The 2025 incident and Daniel's return

---

### Puzzle 5: Cipher Lock - ArcTalon Core
**Difficulty**: Hard | **Type**: Layered Encryption

**Challenge**: Decrypt through 3 layers

**Layer-by-layer solution**:

**Layer 3 (Base64)**: `RUZBU0xJQUY=`
- Decode Base64 ‚Üí `454641534C494146`

**Layer 2 (Hexadecimal)**: `454641534C494146`
- Convert Hex to ASCII ‚Üí `EFASLIAF`

**Layer 1 (Reversed)**: `EFASLIAF`
- Reverse the string ‚Üí `FAILSAFE`

**What you learn**: The core purpose of Protocol ArcTalon

---

### Puzzle 6: The Protocol Key
**Difficulty**: Expert | **Type**: Meta-Puzzle

**Challenge**: What ties all the stories together?

**Solution**: `DUNEWORKS`

**How to solve**: 
- Reflect on all previous puzzles
- They all tell stories about the studio
- The answer is the name of the studio itself

**What you learn**: The complete legacy and Daniel's farewell message

**Reward**: Unlocks the Lore Archive!

## üêõ Troubleshooting

### Server won't start
```bash
# Kill any process on port 3000
lsof -ti:3000 | xargs kill -9

# Try again
npm run dev
```

### Build fails
```bash
# Clean install
rm -rf node_modules package-lock.json
npm install
npm run build
```

### Styling looks wrong
- Hard refresh: `Cmd/Ctrl + Shift + R`
- Clear browser cache
- Make sure you're on latest Chrome/Firefox/Safari

### Terminal input not working
- Click directly on the input field
- Make sure JavaScript is enabled
- Try a different browser
